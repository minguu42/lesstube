# Next App Template

`Next.js` を用いた Web アプリケーション開発用のテンプレートリポジトリ

## 開発環境

- プログラミング言語：TypeScript
- ライブラリ・フレームワーク：React, Next.js
- スタイリング：CSS Modules + Sass
- 状態管理：useContext
- デプロイ：Vercel
- フォーマッタ・リンタ：Prettier, ESLint, stylelint
- テスト：Jest + React Testing Library

### ローカル実行

```bash
make dev
```

### 自動整形

```bash
make fmt
```

### 静的解析

```bash
make lint
```

### テスト

```bash
make test
```
